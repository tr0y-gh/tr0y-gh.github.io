# Resume

This project is my resume and an exercise in minimalist web development. There are no dependencies or build steps but includes modern expectations such as translations, dark and light themes and a (hacky, i know) PDF generator.

If you visit [tr0y-gh.github.io](https://tr0y-gh.github.io) you might notice that the fields email and phone number are redacted. This is to prevent leaking my information and they are added using query parameters. See [this link](https://tr0y-gh.github.io/?email=hello@example.com&phone=%2B46+700+000+000) for an example of what it would look like with placeholder data.
