# Resume

This project is my resume and is hosted on [tr0y-gh.github.io](https://tr0y-gh.github.io). It is an exercise in minimalist web development and there are no dependencies or build steps. All files can be found in [/docs](/docs) and this is the structure:

```
docs/
├── index.html
├── script.js
└── style.css
```

Even if it is simple it has modern features such as translations, dark and light themes and a (hacky, i know) PDF generator.

If you visit using the link above you might notice that the e-mail and phone number are redacted. This is to prevent leaking my information and they are added using query parameters. Use [this link](https://tr0y-gh.github.io/?email=hello@example.com&phone=%2B46+700+000+000) for an example of what it would look like with placeholder data.
